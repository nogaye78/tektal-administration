# TEKTAL – Administration

## Description

Tektal Admin est une application web permettant aux administrateurs de gérer les contenus de la plateforme Tektal.

Elle permet de :
- gérer les chemins (paths)
- gérer les vidéos
- suivre les utilisateurs connectés
- visualiser les données globales via un dashboard

---

## Technologies utilisées

- React
- Vite
- JavaScript

---

## Installation complète

### Cloner le projet

```bash
git clone https://github.com/ton-repository/tektal-admin.git

#  Aller dans le dossier du projet
cd tektal-admin

#  Installer toutes les dépendances
npm install

#  Lancer le projet en développement
npm run dev

#  Ouvrir l'application dans le navigateur

#  Créer un build pour la production
npm run build

#  Prévisualiser le build localement
npm run preview
